# GSS Extract Repository Organization Plan

## Steps

### 1) Inventory the raw data
- Catalog all files in `/data/raw`
- Document file types, sizes, and purposes
- Identify data sources and versions

### 2) Inventory the current set of notebooks
- List all notebooks in `/notebooks`
- Document what each notebook does
- Identify dependencies and relationships between notebooks

### 3) Reorg the notebooks
- Organize notebooks by purpose/functionality
- Establish consistent naming conventions
- Create clear workflow documentation
